What I've Built

Three hands-on projects that demonstrate my backend depth, full-stack range, and DevOps exposure.

#01 Backend · Spring Boot

Expense Tracker Backend

A fully-featured RESTful backend for personal finance management. Users can register, log in securely, and perform complete CRUD operations on their expenses — categorised, filtered, and tracked over time. The application uses Spring Security with JWT tokens so that every API endpoint is protected and only accessible to authenticated users. A password reset flow was implemented via email, making the app production-ready for real users. The entire codebase follows a clean layered architecture (Controller → Service → Repository) with DTOs for clean data contracts and validation annotations to reject bad input before it reaches business logic. All endpoints were tested thoroughly using Postman.

#02 Full-Stack · MERN

Travel Journal Web App

A full-stack web application where users can document and revisit their travel experiences. After signing up and logging in, users can create journal entries for each trip — adding destinations, dates, and personal notes. Entries can be edited and deleted, giving users full ownership of their data. The frontend is built in React, with a Node/Express backend and MongoDB storing all journal data as flexible documents. The entire application was containerised using Docker, meaning it can be launched on any machine with a single command — no manual environment setup needed. This made the app portable and easy to deploy consistently.

#03 DevOps · CI/CD

Automated CI/CD Pipeline — Solar System App

Built a fully automated delivery pipeline so that every code change is tested, scanned, packaged, and deployed without any manual steps. When code is pushed to GitHub, Jenkins (the automation server) automatically picks it up, runs code quality checks via SonarQube, scans the Docker image for security vulnerabilities using Trivy, and pushes the packaged container to a registry. Deployment to an AWS EC2 server is handled by ArgoCD, which watches the repository and automatically keeps the live server in sync with the latest version. This project gave me practical experience in the full software release lifecycle — from code commit to live deployment.
